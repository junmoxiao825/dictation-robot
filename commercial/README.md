## 🚀 自动运行

已添加到Windows启动项，每天开机自动：
1. 生成当日进度报告
2. 弹窗通知进度
3. 保存到 logs/ 目录

**手动运行**:
```bash
cd commercial
python scheduler/auto_run.py
```
