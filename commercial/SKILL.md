## 商业化特性

### 1. 许可证系统
- 基于SHA256的密钥生成
- 线程安全的并发读写
- 90天有效期自动续期提醒

### 2. 功能权限控制
```python
PLANS = {
    'free': {'max_words': 20, 'memory_curve': False, ...},
    'basic': {'max_words': 100, 'memory_curve': True, ...},
    'pro': {'max_words': 999, 'memory_curve': True, 'photo_ocr': True, ...}
}
```

### 3. 支付集成点
- 微信支付：wx.requestPayment
- 支付宝：my.tradePay
- 订阅状态同步到许可证系统
